Online web signup

Online web signup functionality supplied with PortaBilling® enables you to automate the subscription process for new customers to the service. That is, customers can do everything online without requiring any assistance from your administrator.

For backward compatibility, in Dual Version PortaSwitch® newly subscribed customers are created in the source system by default. If you wish to provide online web signup services in the target system and create new customers there, do the following:

  • transfer the signup reseller’s data and that of all their customers to the target system via Porter, and

  • adjust the signup configuration on the source system by defining the IP address of the target system’s SOAP server.
